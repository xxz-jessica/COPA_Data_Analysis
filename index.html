Google Analytics:
Data about the New Website

Google Analytics can automately track website users' log data and represent multiple dimensions and metrics by tables and charts. The data can reveal how users behaved on the website, such as how many pages they visit and the time they spent on a certain page.

Tableau Dashboard:
Analysis of Historical Data

We used Tableau Dashboard to analysis the historical data in the database retrieved in Sep. 2019. There are 5 dimensions we used to analysis the historical data: traffic, engagement, user, forum, and user's feedbacks. Besides conducting descriptive analysis, we also used NLP techniques to learn users' complaints.
