User Behavior Analysis

This section includes the basic analysis of user behavior on the COPA website.

Since the launch of the new COPA site, it has 170k page views with nearly 60% of them are unique page views. Through the line chart, it doesn't change too much day by day with only a small spike over the weekend.

The below graph shows the content drill-down information. The website forum section is the core function of COPA, which shows 40% percent of the user traffic toward the Forum content page. The main site also shows a large number of user traffic, these users are coming in as an organic search where they are browsing through the COPA main website.
